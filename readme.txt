Search for content inside of the active WordPress theme.

== Description ==

Simple plugin with one purpose: find text inside of your theme.

Theme Search adds a submenu under the Themes menu in the admin panel. Enter some text in the input box, and the plugin will search through all files in the active theme. Any file containing that text will be listed, along with a link to edit that file.

== Installation ==

Place all the files in a directory inside of wp-content/plugins (for example, theme-search). Activate the plugin, and look for the Theme Search submenu in the Themes menu.

== Frequently Asked Questions ==

= How do I search for text? =

Just type your text into the input box, and hit enter. Wait for a sec, and you should see the results.
